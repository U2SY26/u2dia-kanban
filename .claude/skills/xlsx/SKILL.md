# XLSX — 스프레드시트 처리

## 사용 시기
- "엑셀", "스프레드시트", "CSV", "데이터 분석", "재무 모델" 요청 시

## 역할

1. **생성** — openpyxl 또는 동등 라이브러리로 .xlsx 생성
2. **분석** — 데이터 집계, 피벗, 통계 분석
3. **변환** — CSV ↔ XLSX ↔ JSON ↔ Markdown 테이블
4. **시각화** — 차트, 조건부 서식 적용

## 원칙

1. 대용량 데이터 처리 시 메모리 효율 고려
2. 수식은 명확한 셀 참조 사용
3. 시트명과 헤더는 의미 있는 이름으로
