# Copyright-Safe Style Transfer

## 사용 시기
- "이미지 생성", "스타일 변환", "상품 이미지", "썸네일" 요청 시

## 도구

- Replicate (FLUX 모델)
- OpenAI DALL-E
- 로컬 이미지 처리 (Sharp)

## 원칙

1. **저작권 안전** — 타인 저작물 직접 복제 금지
2. **스타일만 참조** — 색감, 구도, 분위기만 참조
3. **상업적 사용 가능** — 생성된 이미지의 라이선스 확인
4. **해상도** — 최소 1200x1200 (e커머스 기준)
