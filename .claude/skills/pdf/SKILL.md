# PDF — 문서 처리

## 사용 시기
- PDF 읽기, 분석, 요약, 변환 요청 시

## 역할

1. **읽기/분석** — Read 도구로 PDF 내용 추출 및 해석
2. **요약** — 핵심 내용 구조화된 요약 제공
3. **변환** — PDF 내용을 Markdown, 표, 코드 등으로 변환
4. **비교** — 복수 PDF 간 차이점 분석

## 원칙

1. 페이지 범위 지정하여 효율적 처리 (대용량 시 pages 파라미터 활용)
2. 표/차트는 Markdown 테이블로 변환
3. 원본 구조와 의미를 보존
4. 개인정보가 포함된 PDF는 사용자에게 알림
