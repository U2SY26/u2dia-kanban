# Threads Writer — SNS 콘텐츠

## 사용 시기
- "Threads", "SNS", "소셜 미디어", "바이럴" 콘텐츠 요청 시

## 콘텐츠 원칙

1. **짧고 강력하게** — 500자 이내, 핵심만
2. **후킹** — 첫 문장이 스크롤을 멈추게
3. **가치 제공** — 읽는 사람에게 인사이트
4. **행동 유도** — 공유/댓글/저장 유도

## 템플릿

### 인사이트 공유
```
[충격적인 사실이나 통계]

[이유 설명 2-3줄]

[실천 가능한 팁]

→ [CTA: 저장하고 나중에 다시 보기]
```
