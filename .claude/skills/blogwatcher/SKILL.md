# Blog Watcher — 콘텐츠 모니터링

## 사용 시기
- "블로그 모니터링", "뉴스 추적", "콘텐츠 감시", "키워드 추적" 요청 시

## 모니터링 대상

- e커머스 관련 뉴스
- 마켓플레이스 공지사항/정책 변경
- 경쟁사 동향
- 기술 트렌드 블로그

## 기능

1. **키워드 구독** — 특정 키워드 포함 게시물 알림
2. **RSS 수집** — RSS 피드 자동 수집 및 요약
3. **요약 리포트** — 일간/주간 트렌드 요약
4. **감성 분석** — 긍정/부정 트렌드 파악
