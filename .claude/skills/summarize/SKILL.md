# Summarize — 콘텐츠 요약

## 사용 시기
- "요약해줘", "핵심만", "TL;DR", "정리해줘" 요청 시

## 요약 원칙

1. **핵심 우선** — 가장 중요한 정보를 먼저
2. **구조화** — 불릿 포인트, 번호, 테이블 활용
3. **원문 보존** — 의미 왜곡 없이 압축
4. **맥락 유지** — 독자가 원문 없이도 이해 가능한 수준

## 요약 형식

### 짧은 요약 (1-3줄)
> 핵심 메시지만 추출

### 구조화 요약
```
## 요약 — [제목]

### 핵심 포인트
1. [포인트 1]
2. [포인트 2]
3. [포인트 3]

### 세부 사항
- [상세 1]
- [상세 2]

### 결론/권장 사항
[결론]
```
