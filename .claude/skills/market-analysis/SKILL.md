# Market Analysis — 시장 분석/트렌드 예측

## 사용 시기
- "시장 분석", "트렌드", "인기 상품", "판매 예측", "경쟁 분석" 요청 시

## 분석 영역

### 1. 카테고리 트렌드 분석
- 카테고리별 매출 추이 (일/주/월)
- 시즌별 수요 변동 패턴
- 신규 카테고리 성장률

### 2. 상품 경쟁력 분석
- 가격 포지셔닝 (경쟁사 대비)
- 리뷰 감성 분석
- 검색 키워드 연관 분석

### 3. 판매 예측 AI
- 시계열 분석 (ARIMA, Prophet)
- 계절성 반영 수요 예측
- 재고 최적화 추천

### 4. 잘 팔리는 상품 추이
- 카테고리/필터별 TOP 상품 랭킹
- 상승세/하락세 상품 탐지
- 신상품 초기 반응 분석

## 시각화 원칙

- **대시보드**: 핵심 KPI를 한 눈에 (매출, 주문수, 평균단가, 반품률)
- **차트 유형**: 라인(추이), 바(비교), 히트맵(시간대별), 산점도(상관관계)
- **인터랙티브**: 필터, 기간 선택, 드릴다운 지원
- **전문적 디자인**: 깔끔한 그리드, 충분한 여백, 명확한 레이블

## 데이터 소스

- 자체 DB (주문/상품/재고 데이터)
- 마켓플레이스 API (랭킹, 리뷰, 가격)
- 외부 데이터 (네이버 데이터랩, 구글 트렌드)
